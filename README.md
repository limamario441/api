# API de Seleções ⚽

API REST desenvolvida com Node.js, Express e MySQL para gerenciar seleções de futebol. Permite operações CRUD (Create, Read, Update, Delete) em um banco de dados.

## 🚀 Tecnologias usadas

- Node.js
- Express
- MySQL
- Nodemon (desenvolvimento)
- Arquitetura MVC
